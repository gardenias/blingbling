1. Runtime Version Mamagement

  • SDKMAN - The Software Development Kit Manager,is a tool for managing parallel versions of multiple Software Development Kits on most Unix based systems. It provides a convenient Command Line Interface (CLI) and API for installing, switching, removing and listing Candidates. Formerly known as GVM the Groovy enVironment Manager, it was inspired by the very useful RVM and rbenv tools, used at large by the Ruby community. Get started now! $ curl -s "https://get.sdkman.io" | bash

  • NVP - Node Version Manager - Simple bash script to manage multiple active node.js versions.
